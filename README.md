## ‚öôÔ∏è Overview

**SAMKIEL BOT** is a smart, multi-device WhatsApp bot built with **Node.js** and **Baileys** library.  
It provides advanced moderation, fun interactions, and creative automation features for WhatsApp groups.

Some features include:
- Group admin tools (mute, unmute, kick, tagall)
- AI chat & translation
- Games (Tic-Tac-Toe, Trivia, Hangman)
- Anti-link, anti-badword protection
- Fun utilities (stickers, meme, TTS, truth & dare)
- Media tools (download TikTok, Reels, music)
- Custom text efffects, AI image generation, and more.
